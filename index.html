Code Classes

Web Development

Learn how to build websites with HTML & CSS, once you're able to build a static responsive website, you'll move onto JavaScript to add interactivity to your pages. There will be opportunities to work on a number of different projects during these classes at the end of the projects you'll be able to present your work to club members for feedback.

Register

Game Development

We start off building a game in Scratch which has a drag and drop code area to make it easier for people to learn the basics of programming. Once members build their first game in Scratch, they'll be a choice to use either GameMaker Studio 2 or Godot which are both popular game enginesthat have a lot of available resources to learn online.

Register

Mobile App Development

If you have ever wanted to build your own mobile app or have an idea that you want to build then our mobile app development classes are for you. Compared to the other two classes, we have a different structure for mobile development. Firstly we begin discussing the UI/UX design process then move onto development using Android Dev kit.
